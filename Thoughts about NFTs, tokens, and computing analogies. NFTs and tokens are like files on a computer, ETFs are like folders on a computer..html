Thoughts about Folders and ETFs


Folders serve a huge function in computers.


The metaphor with the computer and files and folders is very interesting and I think very helpful on a larger scale for understanding how important, helpful, and valuable ETFs can be.


Explaining ETFs like folders on a computer is super relatable and understandable. Almost everybody understands how important that is because almost everybody uses folders on their computer.


This computer analogy is worth considering more and can be super resonant, powerful, and helpful.












Below are some more thoughts about how this might be understood in different ways with NFTs, tags, and fractionalized folders.





Tokens, NFTs, and Folders


Sometimes I’ve thought about NFTs and CETFs as computer analogies. NFTs (and tokens in general) are a new file type that can’t be copied (except by whoever owns the root access).


CETFs can offer many tools to manipulate work with NFTs and tokens.


For one example, CETFs provide folder functionality, similar to how a folder in file explorer and Mac finder enable you to put all files in one folder. This is a primitive tool for many other higher level interactions activities and applications.





Folders are an essential primitive of computers. What would we do without folders? It would be difficult to organize our files without folders.


Currently, folder functionality is nascent in community computers, commonly referred to as blockchains.


What will our world look like when folder functionality is mature on community computers?
